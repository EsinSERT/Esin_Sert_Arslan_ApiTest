## Proje Dokümantasyonu

Bu proje, Swagger Petstore API üzerindeki **pet endpoint’leri**
kullanılarak **Rest Assured + TestNG** ile geliştirilmiş
**kod tabanlı API test otomasyonu** örneğidir.

Projede manuel API test aracı (Postman vb.) kullanılmamıştır.
Tüm CRUD senaryoları otomasyon yaklaşımıyla Java üzerinden
gerçekleştirilmiştir.
